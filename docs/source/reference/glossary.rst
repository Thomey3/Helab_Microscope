Glossary
========

.. glossary::

   ADC
      Analog-to-Digital Converter. Converts analog PMT signals to digital values.

   Binning
      Combining adjacent pixels to increase signal at the cost of resolution.

   CUDA
      NVIDIA's parallel computing platform for GPU acceleration.

   DAQ
      Data Acquisition. Hardware for acquiring signals from sensors.

   Descan
      Process of reversing the scan pattern to reconstruct the image.

   Fill Fraction
      The portion of the scan line used for active imaging.

   Galvo
      Galvanometer scanner. A mirror mounted on a galvanometer for beam steering.

   HAL
      Hardware Abstraction Layer. Interfaces that abstract hardware details.

   Hub Device
      Micro-Manager device that manages child devices.

   ILaser
      Interface for laser control operations.

   IClock
      Interface for clock generation operations.

   IScanner
      Interface for scanner control operations.

   IDetector
      Interface for data acquisition operations.

   Micro-Manager
      Open-source software for automated microscope control.

   PMT
      Photomultiplier Tube. A sensitive light detector.

   ROI
      Region of Interest. A sub-region of the full image.

   Resonance Scanner
      High-speed scanner using a resonant galvanometer.

   Result<T>
      std::expected<T, Error> type for error handling.

   Scan Pattern
      The trajectory of the beam across the sample.

   Two-Photon Microscopy
      Fluorescence microscopy using two-photon excitation.
